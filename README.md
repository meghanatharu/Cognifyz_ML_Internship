# Cognifyz Technologies – Machine Learning Internship

This repository contains the machine learning projects completed during my internship at **Cognifyz Technologies**. The projects focus on predictive modeling, recommendation systems, classification, and exploratory data analysis using Python and Scikit-learn.

## Projects

### Task 1 – Restaurant Rating Prediction
- Built a Linear Regression model to predict restaurant ratings.
- Performed data preprocessing, feature engineering, and model evaluation.
- Evaluated using R² Score, Mean Squared Error (MSE), and Root Mean Squared Error (RMSE).

### Task 2 – Restaurant Recommendation System
- Developed a content-based restaurant recommendation system.
- Recommended restaurants based on cuisine, city, and price range.
- Applied similarity-based recommendation techniques.

### Task 3 – Cuisine Classification
- Built a machine learning classification model to predict restaurant cuisines.
- Performed feature encoding, preprocessing, and model evaluation.
- Evaluated using Accuracy, Precision, Recall, and F1-Score.

### Task 4 – Location-Based Analysis
- Performed exploratory data analysis on restaurant locations.
- Identified cities with the highest restaurant density.
- Visualized restaurant ratings, cuisines, and price ranges using Matplotlib and Seaborn.
